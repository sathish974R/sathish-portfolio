# Sathish R — Portfolio

A Java Full Stack Developer portfolio built with React 19 + Vite + Bootstrap 5 + Lucide React.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation & Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
npm run preview
```

## 📦 Dependencies

| Package | Version | Purpose |
|---------|---------|---------|
| react | ^19.2.0 | UI framework |
| react-dom | ^19.2.0 | DOM rendering |
| bootstrap | ^5.3.3 | Responsive grid & utility classes |
| lucide-react | ^0.483.0 | Icons (Code, Mail, Linkedin, etc.) |

## 🛠 Dev Dependencies

- **vite** — fast build tool & dev server
- **@vitejs/plugin-react** — React JSX support
- **eslint** — code linting
